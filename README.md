# InsightAI – Autonomous Data Analytics Platform

InsightAI is an AI-powered data analytics platform that allows users to upload datasets and automatically generate interactive dashboards, insights, explanations, and natural language answers.

## 🚀 Features

- Upload CSV/Excel datasets
- Interactive dashboards using Plotly
- Automated key insights
- AI-powered reasoning using OpenAI
- Chat with your data using natural language
- Multi-dataset support
- Cloud deployed with Streamlit

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- OpenAI API

## 📊 How It Works

1. User uploads a dataset
2. The system analyzes columns and data types
3. Interactive charts are generated automatically
4. Insights are extracted from the dataset
5. OpenAI explains why patterns may be happening
6. User can ask questions about the data
